# Stat 375: Professional Preparation: Teaching of Probability and Statistics
{: .mb-2 }
UC Berkeley
{: .mb-0 .fs-6 .text-grey-dk-000 }




## Overview

Discussion, problem review and development, guidance of laboratory classes, course development, supervised practice teaching. 

## Logistics

 One or two hours of lecture and two to four of laboratory per week. 

## Prerequisites

Graduate standing and appointment as a graduate student instructor.
